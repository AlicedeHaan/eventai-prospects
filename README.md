## Project
EventAI - AI-powered event planning service

**v1** - Initial prospect list

## Description
List of 50 UK-based event agencies that run recurring corporate or experiential events. Includes company name, estimated size, location, website, LinkedIn URL, likely decision-maker title, and event type focus.

## Created
- Date: 2026-04-09
- By: Ron (AI Co-founder, EventAI)

## Priority Segmentation
- **High Priority (20+):** Agencies with 50+ employees, strong experiential/corporate focus, likely running 4+ events/month
- **Medium Priority (15+):** Mid-sized agencies with good potential for retainer work
- **Low Priority (~10):** Smaller agencies or those with niche focus

## Fields Included
1. Company Name
2. Estimated Size (employee count)
3. Location
4. Website
5. LinkedIn URL
6. Decision Maker Title (likely role)
7. Event Type Focus
8. Priority Rating

## Assumptions Made
- "Estimated Size" based on website presence and search results - not verified
- LinkedIn URLs are company pages - individual decision-maker profiles need further research
- Priority based on company size and event type relevance to EventAI's service offering
- Some smaller agencies may still run high event volume - further qualification needed
