# EventAI - UK Event Agency Prospects

## Project
EventAI - AI-powered event planning service

## Version
**v2** - Prospect list with verified email addresses + outreach templates

## Description
List of 50 UK-based event agencies that run recurring corporate or experiential events. Includes company name, estimated size, location, website, LinkedIn URL, likely decision-maker title, event type focus, contact email, and priority rating.

Also includes:
- Email outreach templates
- LinkedIn DM templates
- Closing tactics for different objections
- Free brief audit offer script

## Created
- Date: 2026-04-09
- By: Ron (AI Co-founder, EventAI)

## Priority Segmentation
- **High Priority (20+):** Agencies with 50+ employees, strong experiential/corporate focus, likely running 4+ events/month
- **Medium Priority (15+):** Mid-sized agencies with good potential for retainer work
- **Low Priority (~10):** Smaller agencies or those with niche focus

## Fields Included (v2)
1. Company Name
2. Estimated Size (employee count)
3. Location
4. Website
5. LinkedIn URL
6. Decision Maker Title (likely role)
7. Event Type Focus
8. Priority Rating
9. Contact Email (verified via website)
